Even Spacing Feature - Manual Test Guide

Note: This page is for testing the even spacing snap-to feature. Load this page with the Box Highlight extension enabled.

Test Scenario 1: Horizontal Even Spacing (Between)

  1. Press D to enable drawing mode
  2. Draw two rectangles with a horizontal gap between them:
    • Rectangle A: anywhere on the left side
    • Rectangle B: to the right of A with a noticeable gap
  3. Draw a third rectangle C and drag it between A and B
  4. Expected: When C is positioned to create equal spacing on both sides, it should snap and show a vertical guide line in the gap between A and C (or C and B)

Test Scenario 1b: Horizontal Even Spacing (Right)

  1. With the same two rectangles A and B from above
  2. Draw a third rectangle C and drag it to the RIGHT of B
  3. Expected: When C is positioned so the gap between B and C equals the gap between A and B, it should snap and show a vertical guide line in the gap between B and C
  4. Pattern: [A] gap [B] gap [C] with equal gaps

Test Scenario 1c: Horizontal Even Spacing (Left)

  1. With the same two rectangles A and B
  2. Draw a third rectangle C and drag it to the LEFT of A
  3. Expected: When C is positioned so the gap between C and A equals the gap between A and B, it should snap and show a vertical guide line in the gap between C and A
  4. Pattern: [C] gap [A] gap [B] with equal gaps

Test Scenario 2: Vertical Even Spacing (Between)

  1. Draw two rectangles with a vertical gap:
    • Rectangle A: anywhere at the top
    • Rectangle B: below A with a noticeable gap
  2. Draw a third rectangle C and drag it between A and B
  3. Expected: When positioned for equal spacing, it should snap and show a horizontal guide line in the gap between A and C (or C and B)

Test Scenario 2b: Vertical Even Spacing (Below)

  1. With the same two rectangles A and B from above
  2. Draw a third rectangle C and drag it BELOW B
  3. Expected: When C is positioned so the gap between B and C equals the gap between A and B, it should snap and show a horizontal guide line in the gap between B and C
  4. Pattern: [A] gap [B] gap [C] vertically with equal gaps

Test Scenario 2c: Vertical Even Spacing (Above)

  1. With the same two rectangles A and B
  2. Draw a third rectangle C and drag it ABOVE A
  3. Expected: When C is positioned so the gap between C and A equals the gap between A and B, it should snap and show a horizontal guide line in the gap between C and A
  4. Pattern: [C] gap [A] gap [B] vertically with equal gaps

Test Scenario 3: Duplication with Even Spacing

  1. Draw two rectangles with a gap
  2. Draw a third rectangle anywhere on the page
  3. Hold Alt and click on the third rectangle to duplicate it
  4. Drag the duplicate between the first two rectangles
  5. Expected: Even spacing snap should work during duplication

Test Scenario 4: Repositioning with Even Spacing

  1. Draw three rectangles (any arrangement)
  2. Hold Cmd/Ctrl and click on one rectangle to reposition it
  3. Drag it to create even spacing with other rectangles
  4. Expected: Even spacing snap should work during repositioning

Test Scenario 5: Priority (Edge Alignment vs Even Spacing)

  1. Draw two rectangles with a gap
  2. Position a third rectangle so it would both:
    • Align with the edge of one rectangle
    • Create even spacing between the two rectangles
  3. Expected: Edge alignment should take priority over even spacing (edge guides shown, not spacing guides)

Test Scenario 6: Rectangle Too Large

  1. Draw two rectangles very close together (small gap)
  2. Try to drag a large rectangle between them
  3. Expected: No even spacing snap should occur if the rectangle doesn't fit with room for spacing

What to Look For

  • Spacing guide appearance: Solid colored line (not dashed) using the extension's accent color
  • Spacing guide position: Line appears at the midpoint of the gap
  • Spacing guide length: Line spans only the gap region, not the full viewport
  • Snap behavior: Rectangle snaps smoothly when positioned for even spacing
  • Guide disappears: Spacing guide disappears when you move away from even spacing position
Tip: Use ? to open the shortcuts dialog and verify "Snap to Edges" is enabled in settings.
